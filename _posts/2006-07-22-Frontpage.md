3、Setup：  
```
[x] RubyInstaller
https://rubyinstaller.org/downloads/

[x] Jekyll
gem install jekyll bundler
jekyll -v

[x] gem sources
gem -v
gem sources --add https://gems.ruby-china.com/
gem sources --remove https://rubygems.org/
gem sources -l

[x] Install gems - 国内ruby-china.com镜像服务器
bundle install

[x] Run Application
bundle exec jekyll serve
```
